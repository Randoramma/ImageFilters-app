Welcome to the readme.txt file for the imageFilters app.  
The first phase of the assignment will include :

Layout an Imageview in the center of the screen using autolayout. Give it a ratio of 80% the super views width and height
Create an action controller that has 4 options: camera and 3 seperate filters
Use the UIImagePickerController and its delegate to use the camera to set the imageview’s image
Create a 3 different filter methods on your FilterService class  that filters the imageView’s image with the selected filter.
